### Config
Ohjelman asetukset asetetaan ympäristömuuttujiin.

```
JOONA_PORT = Porttinumero johon joonan webbi serveri bindataan.
JOONA_PETRI_IP = Petrin ip
JOONA_PETRI_PORT = Petrin port

````
